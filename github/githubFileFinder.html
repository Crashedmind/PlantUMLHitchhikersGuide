1. GitHub File Finder¶

Tip

Use GitHub File Finder to search for an icon quickly - directly from GitHub:

  1. Browse to the PlantUML Standard Library repo https://github.com/plantuml/plantuml-stdlib

  2. Press the `t` key to enter GitHub File Finder mode

  3. Type e.g. “lock”, “key”, “user”… or whatever you want to search for

  4. The list of files/icons filenames that contain that text are shown
