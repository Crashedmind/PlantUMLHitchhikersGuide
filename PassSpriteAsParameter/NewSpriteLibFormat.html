4.3. !function and !procedure vs !define and !definelong¶

Per https://plantuml.com/preprocessing#ajlk3nchu0zkka0ybjng:

You should not use !define and !definelong anymore. Use !function, !procedure or variable definition instead.

!define should be replaced by return !function

!definelong should be replaced by !procedure.

The default argument value may help to simplify also definitions There are also now dynamic invocation which may be useful
